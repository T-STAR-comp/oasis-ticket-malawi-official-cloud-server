LOCAL DEV
---------
cd ticket-malawi-cloud-server
SERVE_FRONTEND=true in .env
npm run dev

PRODUCTION BUILD
----------------
npm run build:all
(requires main app as a sibling folder)

CPANEL / SPACESHIP
------------------
Upload both repos side by side, e.g.:
  oasis-ticket-malawi-official-cloud-server/
  oasis-ticket-malawi-official-main-app/

In cloud-server Terminal:
  npm install
  npm run build:all
  (create .env from .env.example)
  Restart Node app (startup file: server.cjs)
